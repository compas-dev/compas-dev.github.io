Core Extensions

Core COMPAS extension for computational research in Architecture, Engineering, Fabrication, and Construction.

compas_fab

Robotic fabrication package providing interface to ROS.

compas_vol

Volumetric modelling of 3D geometry using signed distance functions.

compas_slicer

Slicing tools for additive manufacturing processes.

compas_rrc

Online non-realtime control for ABB robots over a simple-to-use Python API.

compas_fea

Finite Element Analysis using Ababqus, Ansys, or OpenSEES.

compas_fem

Implementation of the Finite Element Method based on Fenics/Dolfin.

compas_assembly

Data structures for discrete element assemblies.

compas_rbe

Rigid Block Equilibrium analysis of discrete element alssemblies.

compas_tia

Topological Interlocking Analysis for complex discrete element assemblies.

compas_cloud

RPC using websockets for bi-directional client-server communication.

compas_viewers

Standalone viewers for basic 3D visualisation.

Extensions

Extension developed by users related to their research, projects, or other computational work.

compas_singular

Topology finding of singularities in patterns.

compas_skeleton

Skeleton-based volumetric and surface mesh modeling.

compas_ags

Algebraic graphic statics for two-dimensional structures.

compas_tna

Form Finding of funicular networks using TNA.

compas_3gs

3D Graphic statics using polyhedral form and force diagrams.

compas_cem

Combinatorial Equilibrium Modelling.
