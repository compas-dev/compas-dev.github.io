Extensions

The compas-dev team, with support from the NCCR "Digital Fabrication" at ETH Zurich, develops core COMPAS extension for research and collaboration in Architecture, Engineering, Fabrication, and Construction.

compas_fab

Robotic fabrication package providing interface to ROS.

compas_fea

Finite Element Analysis using Ababqus, Ansys, or OpenSEES.

compas_fem

Implementation of the Finite Element Method based on Fenics/Dolfin.

compas_vol

Volumetric modelling of 3D geometry using signed distance functions.

compas_rbe

Rigid Block Equilibrium analysis of discrete element alssemblies.

compas_tia

Topological Interlocking Analysis for complex discrete element assemblies.

compas_rrc

Online non-realtime control for ABB robots over a simple-to-use Python API.

compas_am

Tools for additive manufacturing processes.

compas_assembly

Data structures for discrete element assemblies.

compas_cloud

RPC using websockets for bi-directional client-server communication.

Plugins

Plugins are packages developed and maintained by compas-dev that provide missing functionality of COMPAS core or faster alternatives to (pure Python) COMPAS core algorithms.

compas_triangle

2D Mesh generation and refinement using Triangle.

compas_libigl

Geometry processing using libigl.

compas_cgal

Polygon Mesh Processing using CGAL.

compas_gmsh

2D and 3D meshing using GMSH.

Contributed Packages

Many other packages have been contributed by various users and user groups, related to their research, projects, or other computational work.

compas_skeleton

Skeleton-based volumetric and surface mesh modeling.

compas_tna

Form Finding of funicular networks using TNA.

compas_singular

Topology finding of singularities in patterns.

compas_3gs

3D Graphic statics using polyhedral form and force diagrams.

compas_ags

Algebraic graphic statics for two-dimensional structures.
