Extensions

The NCCR "Digital Fabrication" develops core COMPAS extension for Architecture, Engineering, Fabrication, and Construction (AEFC) research and collaboration.

compas_fab

Robotic fabrication package providing interface to ROS.

compas_fea

Finite Element Analysis using Ababqus, Ansys, or OpenSEES.

compas_fem

Implementation of the Finite Element Method based on OpenSEES and Dolfin.

compas_vol

Volumetric modelling of 3D geometry using signed distance functions.

compas_am

Additive manufacturing.

compas_triangle

2D Mesh generation and refinement using Triangle.

compas_pmp

Polygon Mesh Processing using CGAL.

compas_tia

Topological Interlocking Analysis for complex discrete element assemblies.
