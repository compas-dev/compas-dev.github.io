Research & Collaboration

COMPAS is an open source, python-based framework for computational research and collaboration in architecture, engineering, fabrication and construction (AEFC).

Core Functionality

The main library of COMPAS provides flexible data structures, a geometry processing library, robot fundamentals, numerical solvers, and various other components for as a base framework for computational AEFC research.

CAD Integration

CAD tools are everywhere in AEFC, both in research and in practice. COMPAS provides a consistent interface for working with CAD geometry, and for visualising and intercating with COMPAS geometry and data structures in various softwares.

Rhino


import compas

from compas.datastructures import Mesh
from compas_rhino.artists import MeshArtist

mesh = Mesh.from_obj(compas.get('tubemesh.obj'))

artist = MeshArtist(mesh)
artist.draw_vertices()
artist.draw_faces()
artist.draw_edges()

Exchange

To simplify the data exchange between AEFC software, COMPAS supports commonly used file formats.

Communication

COMPAS provides a flexible data serialisation mechanism, and implements Remote Procedure Calls, to facilitate the communication between different softwares, and processes. 


from compas.rpc import Proxy
proxy = Proxy('compas.numerical')

origin, axes, values = proxy.pca_numpy(...)
